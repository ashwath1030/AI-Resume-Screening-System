# AI Resume Screening System

An AI-powered Resume Screening System built with FastAPI and React.

## Features

- User Authentication (Admin, HR, Recruiter)
- Resume Upload
- Resume Parsing
- AI Resume Ranking
- Candidate Management
- Job Management
- Dashboard Analytics
- PDF Report Generation
- Excel Report Generation
- Email Notifications

## Tech Stack

### Frontend
- React
- Material UI
- Axios

### Backend
- FastAPI
- SQLAlchemy
- PostgreSQL (NeonDB)

### AI
- Sentence Transformers
- Resume Parsing
- Semantic Matching

## Installation

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
